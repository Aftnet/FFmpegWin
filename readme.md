# FFmpeg Windows builds

Visual Studio builds of the FFMPEG libraries as shared DLLs, unmodified from their source repo.

## Supported platforms

- Windows 10 UWP (x86, x64, arm, arm64)
- Windows 10 Desktop (x86, x64, arm64)
